## imgpreview
Preview images from the terminal.

## Installation
	git clone https://github.com/mattlawer/imgpreview.git
	cd imgpreview
	make
	make install

## Usage
	Usage : imgpreview [-a -g -n] <image>
		-a : use ascii
		-g : use grayscale
		-n : no colors
	
	example:
		imgpreview -an dancing-cat.png
